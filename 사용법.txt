ㅎㅇㅎㅇ 내가 디자인 감각은 좀 후달리고 
프론트쪽은 공부가 부족하다 그러니 이해좀여
맨 위에 메뉴에서 마우스가 손가락 모양으로 바뀌어야 클릭되는 거임!
색깔만 바뀐다고 클릭되는게 아님

1. 시작하는법
- 사실 .html 파일 아무거나 눌러도 된다.
- 후에 서버에 올릴 때는 index.html을 홈페이지로 사용할 예정
- 뭔소리인지 모르겠으면 index.html 더블클릭해서 실행

2. 사용자 가이드
- 대충 어떤 메뉴가 어떤 일을 하는지 적은 건데 사실 난 잘 몰라서 걍 적음

3. 선수 세부 기록
- 선수별 데이터를 기록 가능
- 선수정보추가 버튼을 누르면 표의 행이 늘어난다
- 우측의 '삭제' 체크박스를 누르면 삭제된다.
- 입력하고 싶은 칸을 클릭하면 텍스트 입력가능하게 만들어놨음

4. 히트맵 분석
- 기본은 우리팀(파란색)으로 설정되어있음
- 맵 위에서 원하는 위치에서 마우스 좌클릭하면 파란색 원이 나타남
- 생성된 원을 우클릭하면 삭제됨
- 상대팀을 누르면 붉은색 원이 생성되며, 나머지 동일

5. 양팀 라인업
- 버튼은 그냥 재활용했고 별 기능 없다
- 왼쪽 표에 우리팀, 오른쪽 표에 상대팀 정보 입력
- 전술에 따라 위치가 달라질 수 있을 거 같아서 포지션과 이미지는 연동ㄴㄴ
- 필드에서 원하는 위치를 클릭하면 포지션을 입력해주세요!! 라는 창이 뜸
- 거기에 원하는 포지션(예:LM)이런식으로 적으면 입력받은 포지션을 기준으로 마크가 생성됨
- 사실 이걸 제일 잘 못 만든 거 같음
